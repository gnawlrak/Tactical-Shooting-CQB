# AI 攻击意识增强与声音感知系统

## 🎯 优化目标

1. **增强攻击意识** - 解决"玩家到脸上才开一枪"的问题
2. **添加声音感知** - AI 能根据脚步、枪声判断玩家位置
3. **预判架枪** - AI 利用声音信息提前占据有利位置

---

## ✨ 新增功能

### 1. **声音感知系统**

#### 1.1 声音类型与传播

| 声音类型 | 传播距离 | 置信度衰减 | AI 响应 |
|---------|---------|-----------|--------|
| **枪声** | 80m (×2.0) | -0.2/秒 | 立即警觉，寻找掩体 |
| **脚步声** | 28-40m (姿态相关) | -0.2/秒 | 开始调查，朝声音方向看 |
| **换弹声** | 32m (×0.8) | -0.2/秒 | 可能选择进攻 |
| **落地声** | 60m (×1.5) | -0.2/秒 | 警觉并定位 |

#### 1.2 声音置信度计算

```gdscript
置信度 = 1.0 - (距离 / 听力范围)

# 示例
玩家在 20m 外开枪：
置信度 = 1.0 - (20 / 80) = 0.75  # 很高

玩家在 35m 外走路：
置信度 = 1.0 - (35 / 40) = 0.125  # 很低
```

#### 1.3 AI 响应逻辑

```gdscript
听到枪声:
    ✓ 立即进入 DEFEND 状态
    ✓ 寻找最近掩体
    ✓ 通知附近敌人（50m 内）
    ✓ 设置 ambush_timer 3-6 秒

听到脚步声/落地声:
    ✓ 记录最后声音位置
    ✓ 设置 investigation_timer 2-4 秒
    ✓ 朝声音方向转头
    ✓ 开始调查

听到换弹声:
    ✓ 如果距离<20m，立即冲锋
    ✓ is_rushing = true
    ✓ current_state = CHASE
```

### 2. **预判架枪系统**

#### 2.1 工作流程

```
1. 听到声音 → 记录位置
   ↓
2. 预测玩家移动方向
   (target_pos - sound_pos).normalized()
   ↓
3. 选择能覆盖预测区域的掩体
   - 射界良好 (+30 分)
   - 距离适中 15-30m (+20 分)
   ↓
4. 移动到掩体并架枪
   ✓ 占据有利位置
   ✓ 提前瞄准预测点
   ✓ 等待玩家出现
```

#### 2.2 架枪位置评分

```gdscript
func evaluate_cover_for_angle(cover_pos, predict_pos):
    score = 0
    
    # 1. 检查射界（能否看到预测区域）
    if has_good_firing_angle(cover_pos):
        score += 30
    
    # 2. 距离预测点的距离（中等最好）
    dist = distance(cover_pos, predict_pos)
    if 15 < dist < 30:
        score += 20
    
    return score
```

### 3. **攻击意识增强**

#### 3.1 近距离快速反应 (<15m)

**旧行为**:
```
看到玩家 → 等待 burst_pause 结束 → 开火
延迟：~1-2 秒
```

**新行为**:
```
看到玩家 → 立即开火 → 连续射击
延迟：<0.1 秒
```

**代码实现**:
```gdscript
if dist < 15.0:
    if fire_cooldown <= 0:
        shoot()
        # 减少犹豫时间
        burst_pause = max(0, burst_pause - delta * 0.5)
```

#### 3.2 中距离正常交火 (15-20m)

```gdscript
if dist < 20.0:
    # 增加射击次数
    burst_count = randi_range(3, 6)  # 原 2-4
    # 缩短间隔
    burst_pause = randf_range(1.0, 2.0)  # 原 2-4
```

#### 3.3 远距离精确点射 (>20m)

```gdscript
# 保持游击战术
burst_count = randi_range(2, 4)
burst_pause = randf_range(2.0, 4.0)
```

## ⚖️ 平衡性设计

### 声音传播限制

1. **距离衰减** - 超过听力范围完全听不到
2. **置信度衰减** - 声音随时间变得不可靠
3. **障碍物影响** - 墙壁会减少可听距离（未来扩展）

### AI 响应限制

1. **不是透视** - AI 只能听到声音，不能穿墙看到玩家
2. **需要确认** - 单一次声音不会让 AI 完全确定位置
3. **会犯错** - 低置信度时 AI 可能判断错误

## 💡 设计哲学

> **"信息战是现代战斗的核心"**

真实的特种作战中，士兵会通过声音判断敌人位置、数量和意图。现在的 AI 具备了这种能力：

1. **听觉感知** - 能听到玩家的行动
2. **信息处理** - 分析声音类型和距离
3. **战术决策** - 根据声音采取相应行动
4. **团队协作** - 分享听到的信息

这让玩家必须像真正的特种兵一样思考：
- 我的每个动作都会暴露信息
- 敌人会根据这些信息做出反应
- 我需要控制自己的存在感
